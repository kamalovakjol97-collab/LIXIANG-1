# Диагностика проблем с белым экраном

## Возможные причины

### 1. Ошибки JavaScript в консоли браузера
**Проверка:**
- Откройте сайт в браузере
- Нажмите F12 (или правый клик → Inspect)
- Перейдите на вкладку **Console**
- Проверьте наличие красных ошибок

**Решение:**
- Скопируйте текст ошибки
- Проверьте, что все импорты корректны
- Убедитесь, что все файлы существуют

### 2. Проблемы с переменными окружения
**Проверка в Netlify:**
1. Site settings → Environment variables
2. Убедитесь, что указаны:
   - `VITE_SUPABASE_URL` = `https://lexofzwhenyauodouqxz.supabase.co`
   - `VITE_SUPABASE_ANON_KEY` = `sb_publishable_xH9vTWI6N6QM6QtwjyokYQ_Sk9XZCQF`

**Решение:**
- Добавьте переменные, если их нет
- Перезапустите деплой (Retry deploy)

### 3. Ошибки сборки
**Проверка:**
- В Netlify перейдите в Deploys
- Откройте последний деплой
- Проверьте логи сборки на наличие ошибок

**Решение:**
- Исправьте ошибки в коде
- Убедитесь, что `npm run build` выполняется успешно локально

### 4. Проблемы с роутингом
**Проверка:**
- Убедитесь, что файл `_redirects` существует в корне проекта
- Проверьте `netlify.toml` на корректность настроек

### 5. Ошибки в компонентах
**Проверка:**
- Откройте консоль браузера
- Ищите ошибки типа "Cannot read property of undefined"
- Проверьте, что все хуки используются корректно

## Быстрая проверка

1. **Локально:**
   ```bash
   npm install
   npm run build
   npm run preview
   ```
   Если локально работает, проблема в деплое.

2. **В Netlify:**
   - Проверьте логи деплоя
   - Проверьте переменные окружения
   - Проверьте настройки сборки

3. **В браузере:**
   - Откройте консоль (F12)
   - Проверьте вкладку Network на наличие ошибок загрузки
   - Проверьте вкладку Console на наличие ошибок JavaScript
